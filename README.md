# 🏏 IPL Analytics Platform

![Banner](dashboard/assets/banner.png)

## 📌 Overview

IPL Analytics Platform is an end-to-end Data Analytics project built using Python, SQL and Streamlit.

The project analyzes IPL matches from **2008–2024**, transforming raw match and ball-by-ball data into interactive dashboards, statistical insights and business reports.

---

## 🚀 Features

- Data Cleaning using Pandas
- Exploratory Data Analysis (EDA)
- Player Performance Analysis
- Team Performance Analysis
- Venue Analysis
- SQL Analytics
- Interactive Streamlit Dashboard
- Downloadable Reports
- Business Insights

## 📈 Key Insights

- Mumbai Indians and Chennai Super Kings have consistently been among the most successful franchises.
- Toss decisions show varying impact depending on venue and season.
- Batting-first and chasing strategies differ significantly across stadiums.
- Orange Cap and Purple Cap analyses highlight player consistency across seasons.
- Venue-specific trends reveal differences in scoring and match outcomes.

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd ipl-data-analyst-project
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
streamlit run dashboard/app.py
```
